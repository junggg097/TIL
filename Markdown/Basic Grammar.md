# 마크다운을 연습해보자  

마크다운은 대표적인 마크업 언이 이다. 
#은 문단 제목, 문단 구분, 그 아래 작성되는 그냥 글은 문단 내용

두번째 문단을 만들어 보자 엔터 두번해야 새로운 문단이 된다.

세번째 문단은 이렇다. 스페이스 두개 후 엔터  
해도 줄은 바뀐다.

엔터 두 번 -> 문단 구분  
스페이스 두 번,  엔터 한 번 -> 줄 바꿈
